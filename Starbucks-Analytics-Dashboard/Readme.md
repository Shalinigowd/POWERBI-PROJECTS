# Starbucks Analytics Dashboard

## Project Overview

This Power BI dashboard analyzes Starbucks beverage nutritional data to provide insights into calories, caffeine, sugar, and protein content across different beverage categories.

## Objectives

* Analyze beverage nutritional composition
* Identify high and low caffeine beverages
* Compare calories, sugar, and protein across categories
* Support data-driven health and product insights

## Dashboard Features

* Interactive slicers and filters
* KPI Cards
* Beverage category distribution analysis
* Average caffeine and calorie comparison
* Sugar vs Calories correlation analysis
* Top 5 highest caffeine beverages
* Business insights and recommendations

## Key Metrics

* Total Beverages
* Average Calories
* Average Sugar
* Average Caffeine
* Average Protein

* Data Visualization
